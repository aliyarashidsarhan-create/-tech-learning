Web Development:
 refers to the process of building, creating, and maintaining websites and web applications that run on the internet.

1. Front-End Development (Client-Side)

This is what users see and interact with directly.
Front-end developers use:

HTML – structure of the website

CSS – styling and layout

JavaScript – interactivity and dynamic features

2. Back-End Development (Server-Side)

This is the part users don’t see. It handles:

Databases

Servers

Application logic

Common languages & tools:

Node.js

Python (Django/Flask)

PHP

Java

SQL databases
